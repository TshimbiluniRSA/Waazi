# Waazi Lite (React + Vite Mock Prototype)

This is a frontend-only prototype built with React and Vite using mock data.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```

## Notes

- No backend integration yet.
- Risk scoring, recommendations, FAQ, and health facilities are mock/demo data in `src/mockData.js`.
